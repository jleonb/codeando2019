Mi primera página

En este curso obtendrás los conocimientos básicos para comenzar a crear tus propios proyectos. A estructurar tus sitios web con HTML y a darle estilo con CSS.

Aprenderás a utilizar Bootstrap, uno de los frameworks más populares para el desarrollo de sitios web responsivos. Implementarás librerías JavaScript para darle dimanismo a tus proyectos con animaciones, galerías, efectos, etc.

Veremos cuáles son los pasos necesarios para publicar tu sitio, lo que es un hosting, un dominio y todos los pasos necesarios para tener tu sitio online.

Incluye:

  • HTML
  • CSS
  • Librerías JavaScript
  • Bootstrap
  • Publicar tu sitio

No necesitas requisitos previos, solo debes traer tu notebook.

Mi primera web app

Ruby on Rails es un framework de desarrollo web basado en el lenguaje de programación Ruby, posee un enorme ecosistema de librerías para resolver casi cualqueir necesidad.

Incluye:

  • Patrón de diseño Modelo Vista Controlador
  • Estructura de un proyecto Rails
  • Crear y actualizar una base de datos
  • Agregar funcionalidad a la App mediante gemas
  • Git - GitHub
  • Publicar tu App

Es aconsejable que alumno que tome este curso tenga conocimientos previos de HTML y CSS.

JavaScript

JavaScript es un lenguaje de programación que permite agregar contenido dinámico e interactivo a tus sitios web. Podrás animar elementos, controlar archivos multimedia y crear respuestas interactivas a eventos (click de botones, relleno de formulario, scrolling, etc)

Incluye:

  • Consola Web
  • Manejo del DOM
  • Datos y control de flujo (if, array, while, for)
  • Funciones
  • Eventos
  • JQuery
  • Animaciones
  • Validación de Formularios
  • Mini Aplicación JS

Es aconsejable que los alumnos que tomen este curso, tengan conocimientos previos de HTML y CSS.
